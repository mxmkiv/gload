# gload

Утилита для нагрузочного тестирования HTTP, написанная на Go. Аналог [k6](https://k6.io).

## Возможности

- Конкурентные виртуальные пользователи с общим пулом соединений
- Прогресс-бар в реальном времени
- Подробные результаты: перцентили латентности (p50/p95/p99), RPS, статистика по кодам ответов

## Установка

```bash
git clone https://github.com/mxmkiv/gload.git
cd gload
go install ./cmd/...
```

После установки `gload` доступен глобально через `$GOPATH/bin`.

## Конфигурация

Создайте JSON-файл конфигурации:

```json
{
  "source": "https://example.com",
  "UVs": 100,
  "time": "30s"
}
```

| Поле     | Тип    | Описание                                                                 |
|----------|--------|--------------------------------------------------------------------------|
| `source` | string | Целевой URL                                                              |
| `UVs`    | int    | Количество конкурентных виртуальных пользователей                        |
| `time`   | string | Длительность теста. Формат: `10s`, `1m`, `1m30s` или число — количество секунд (`30` = 30s) |

## Запуск

```bash
gload --config config.json
```

Если флаг `--config` не указан, программа ищет `config.json` в текущей директории.
